### Import the Trevas TS engine

```bash
yarn add @inseefr/trevas
```

### Call the Trevas TS engine

```typescript
import { interpret, type Bindings } from 'trevas';

const bindings: Bindings = { k: 'v' };

const result = interpret('VTL expression', bindings);
```
